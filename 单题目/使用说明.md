# 积木编程AI出题工具 - 使用说明

## 快速开始

### 方法1：直接打开（推荐）
双击 `AI出题工具.html` 文件即可使用

### 方法2：本地服务器
双击 `启动工具.bat`，自动打开浏览器，需要Python基础

## 首次配置

### 1. 输入Base URL
- Claude官方：`https://api.anthropic.com`
- 国内中转：根据服务商提供的地址

### 2. 输入API Key
- 从API服务商获取
- 会自动保存在本地浏览器

### 3. 开始出题
- 输入主题描述
- 选择难度和题型
- 点击"生成题目"

## 常见问题

### Q: 提示"failed to fetch"
**A:** 检查以下几点：
1. Base URL是否正确
2. API Key是否有效
3. 网络是否正常
4. 是否需要代理

### Q: 生成的代码不正确
**A:** 修改 `系统提示词配置.md`，调整语法规范

### Q: 无法保存配置
**A:** 浏览器可能禁用了本地存储，请检查浏览器设置

### Q: 双击HTML无法打开
**A:**
1. 右键 → 打开方式 → 选择浏览器
2. 或使用 `启动工具.bat`
