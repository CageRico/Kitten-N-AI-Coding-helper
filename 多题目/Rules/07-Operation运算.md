# 编程语言规范文档：运算分类 (Operation)

**分类概述**： 顾名思义，这个部分的积木多涉及运算任务，由于我们的规则编程是面向少儿，因此多为小学中学数学类似运算，此外，也涉及很多**变量操作**，需要与变量类的积木块关联起来考虑，这部分多为单句命令。其中涉及很多**判断类**积木，这类的特点是会返回Bool类型，True或False。

**积木名称规则**: 积木名称是严格的，虽然为自然语言，但每个积木的名称是**严格固定**的，也即下面的每块积木的名称。

**积木语法规则**: 每块积木的运行语法也是有**严格**要求的，其包含的括号如:(){}、以及=>等都是严格的，其次，分号;也是严格的。

**调用规则**: 当你需要引用积木时，需要考虑该积木的含义和作用，调用时必须严格调用其**代码语法**，这个是**严格的**，代码语法中一些非参数部分是不允许改的。

## 1. 积木名称：五则运算
* 积木ID：operation_normal
* 功能描述：两个指定的变量进行指定的运算。
* 参数定义：
  * 变量1(x1): 数字类型，可以由用户定义。
  * 变量2(x2): 数字类型，可以由用户定义角色。
  * 运算类型(y): 分别为+-*/^
* 代码语法：
  x1 y x2;
* 代码示例: 
  * 1加1：1 + 1;
  * 1减1：1 - 1;
  * 1乘1：1 * 1;
  * 1除1：1 / 1;
  * 1的2次方：1 ^ 2;

## 2. 积木名称：随机整数
* 积木ID：operation_randint
* 功能描述：取出指定的两个数之间的随机整数。
* 参数定义：
  * 上边界1(x1): 数字类型，可以由用户定义。
  * 下边界2(x2): 数字类型，可以由用户定义角色。
* 代码语法：
  随机整数(x1, x2);
* 代码示例: 
  * 取0-5之间的一个随机的整数：随机整数(0, 5);

## 3. 积木名称：数值判断（Bool）（判断类）
* 积木ID：operation_bool
* 功能描述：判断两个数的大小关系，一共6类关系：大于、小于、不等于、等于、大于等于、小于等于。
* 参数定义：
  * 变量1(x1): 数字类型，可以由用户定义。
  * 变量2(x2): 数字类型，可以由用户定义角色。
  * 判断关系(yy): 等于是===、不等于是!==、大于是>、小于是<、小于等于是<=、大于等于是>=
* 代码语法：
  x1 yy x2;
* 代码示例: 
  * 判断0不等于0：0 !== 0;
  * 判断0等于0：0 === 0;
  * 判断0小于0：0 < 0;
  * 判断0小于等于0：0 <= 0;

## 4. 积木名称：且或判断（判断类）
* 积木ID：operation_and_or
* 功能描述：判断两个条件同时成立或只成立一个，可以理解为and、or。
* 参数定义：
  * 条件1(x1): 子代码，可以由用户定义。
  * 条件2(x2): 子代码，可以由用户定义角色。
  * 关系(yy): 且&&、或||
* 代码语法：
  x1 yy x2;
* 代码示例: 
  * 0小于等于0和0大于0同时成立：0 <= 0 && 0 > 0;
  * 0小于等于0或0大于0：0 <= 0 || 0 > 0;

## 5. 积木名称：不成立判断（判断类）
* 积木ID：operation_not
* 功能描述：某个条件不成立则为True。
* 参数定义：
  * 条件(x1): 子代码，可以由用户定义。
* 代码语法：
  ! x1;

## 6. 积木名称：四舍五入
* 积木ID：operation_round
* 功能描述: 对给定的数进行四舍五入操作。
* 参数定义：
  * 数字(x1): 数值，可以由用户定义。
  * 操作(yy): 字符串，只可以是“四舍五入”、“向上舍入”、“向下舍入”。
* 代码语法：
  舍入运算("yy", x1);
* 代码示例: 
  * 对3.1四舍五入：舍入运算("四舍五入", 3.1);

## 6. 积木名称：取余数
* 积木ID：operation_left
* 功能描述: 取两个给定的数相除的余数。
* 参数定义：
  * 被除数(x1): 数值，可以由用户定义。
  * 除数(x2): 数值，可自定义。
* 代码语法：
  余数运算(x1, x2);
* 代码示例: 
  * 对64/10取余数：余数运算(64, 10);


## 7. 积木名称：整除判断（判断类）
* 积木ID：operation_zhengchu
* 功能描述: 判断两个数是否可以整除
* 参数定义：
  * 被除数(x1): 数值，可以由用户定义。
  * 除数(x2): 数值，可自定义。
* 代码语法：
  整除判断(x1, x2);
* 代码示例: 
  * 判断9和3可否整除：整除判断(9, 3);

## 8. 积木名称：数值判断（判断类）
* 积木ID：operation_num_judge
* 功能描述: 判断某个数的性质，是否是正数、负数等。
* 参数定义：
  * 数值(x1): 数值，可以由用户定义。
  * 类型(yy): 字符串，只可以是“质数”、“整数”、“合数”、“奇数”、“偶数”、“正数”、“负数”。
* 代码语法：
  数值判断(x1, "yy");
* 代码示例: 
  * 判断9是否为质数：数值判断(9, "质数");

## 9. 积木名称：三角函数
* 积木ID：operation_sin
* 功能描述: 返回某个三角函数值，支持sin、cos、tan等。
* 参数定义：
  * 角度(x1): 数值，可以由用户定义，单位为度。
  * 类型(yy): 字符串，只可以是“sin”、“cos”、“tan”、“asin”、“acos”、“atan”。
* 代码语法：
  三角函数("yy", x1);
* 代码示例: 
  * 获取sin45°的值：三角函数("sin", 45);

## 10. 积木名称：其他数值运算
* 积木ID：operation_other
* 功能描述: 对某个数值进行特殊运算。
* 参数定义：
  * 数字(x1): 数值，可以由用户定义。
  * 运算(yy): 字符串，只可以是“算术平方根”、“绝对值”、“-”（取负数）、“ln”、“log10”、“e^”、“10^”。
* 代码语法：
  科学计算("yy", x1);
* 代码示例: 
  * 获取0的算术平方根：科学计算("算术平方根", 0);

## 11. 积木名称：字符串拼接
* 积木ID：operation_merge
* 功能描述: 对两个字符串拼接。
* 参数定义：
  * 字符串1(x1): 字符串，可以由用户定义。
  * 字符串2(x2): 字符串，可以由用户定义。
* 代码语法：
  连接字符串("x1", "x2");

## 12. 积木名称：字符串切片
* 积木ID：operation_index
* 功能描述: 取出某个字符串的某位的字符
* 参数定义：
  * 字符串1(x1): 字符串，可以由用户定义。
  * 头索引(y1): 数字类型，可以由用户定义。
  * 尾索引(y2): 数字类型，可以由用户定义。
* 代码语法：
  选择字符串("x1", y1, y2);
* 代码示例: 
  * 字符串abc的第一个字符：选择字符串("abc", 1, 1);
  * 字符串abc的前两个字符串：选择字符串("abc", 1, 2);

## 13. 积木名称：字符串切片
* 积木ID：operation_len
* 功能描述: 返回某个字符串的长度
* 参数定义：
  * 字符串1(x1): 字符串，可以由用户定义。
* 代码语法：
  字符串长度("x1");
* 代码示例: 
  * 字符串abc的长度：字符串长度("abc");

## 14. 积木名称：是否包含（判断类）
* 积木ID：operation_in
* 功能描述: 判断某个字符串中是否有某个字符串或字符
* 参数定义：
  * 字符串(x1): 字符串，可以由用户定义。
  * 小字符串(x2): 字符串，可以由用户定义。
* 代码语法：
  字符串包含判断("x1", "x2");
* 代码示例: 
  * 字符串abc是否包含a：字符串包含判断("abc", "a");


## 15. 积木名称：把字符串分开为列表
* 积木ID：operation_split
* 功能描述: 将字符串按照某个字符分割为列表，类似于split。
* 参数定义：
  * 字符串(x1): 字符串，可以由用户定义。
  * 分割符(x2): 字符串，可以由用户定义。
* 代码语法：
  分开字符串("x1", "y");
* 代码示例: 
  * 字符串a,b,c按照,分割为列表：分开字符串("a,b,c", ",");

## 15. 积木名称：转换数据类型
* 积木ID：operation_change
* 功能描述: 强制转换某个类型。
* 参数定义：
  * 原类型(x1): 对应类型，可以由用户定义。
  * 新类型(x2): 字符串，只可以是“字符串”、“数值”、“布尔”。
* 代码语法：
  类型转换("x1", "x2");
* 代码示例: 
  * 数字123转换为字符串类型：类型转换("123", "字符串");
